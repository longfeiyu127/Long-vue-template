# Intro
A Vue.js template that can support more than 100 thousand lines of code in our business, I hope it can help you too~

Status: building...

## Docs
- [tutorial](/docs/tutorial.md)

## Start
run followed code in your iterm:
```
npm i
npm run dev
```

## Commands
- `npm run dev`
- `npm run start`
- `npm run build`
- `npm run mock`
- `npm run analyz`


## Config

UI框架 -- VUX
数据管理 -- vuex
路由管理 -- vue-router
适配方案 -- lib-flexible
数据请求 -- axios
